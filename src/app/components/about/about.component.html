About

This project was built within a Master Thesis conducted at the iwb at Technical University Munich. The goal was to create a pipeline for researchers to train a convolutional neural network on labeled images to detect welding defects and extract their metrics. The iwb had welding images that were taken from builds created through an additive manufacturing process such as friction stir welding (FSW) or selective laser melting (SLM). The images included the build and defects as well as a measurement line and text.

The challenges included:

  • recognition of measurement text
  • detection and localization of measurement lines
  • segmentation of welding defects

For the creation of a software-solution, following elements were used:

  • Angular for frontend
  • Flask for backend
  • Detectron2 framework for recognition/segmentation tasks
  • Mask R-CNN network for defect segmentation
  • EasyOCR package for text recogntition
  • Faster R-CNN network for line detection

The pipeline was setup in a way that the user can train locally on labeled images. In the thesis, a process is also explained where the user can label his own images. Then, with the updated model, the user can perform the inference step, which extracts the text and detects the measurement lines. With this information, the segmentation of the welding defects is put into context accordingly. The image-selection process is done on the frontend. The images are then sent to the python-based backend, where the inference step takes place. The result with the images and metrics are then sent back to the frontend to display them.
